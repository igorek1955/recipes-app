# Recipes for Android
using spoonacular API. 

#Functionality
- search recipes
- save recipes
- have a chuckle at food jokes

# Features
- retrofit REST
- Dagger-Hilt - DI
- DataStore - shared preferences
- Motion Layout
- Room DB for persistance
- Lazy load with shimmer effect
- Night Mode support
- LiveData 
- Data Binding
- View Binding
- ViewPager2

# Setup
Before usage insert your API key into Constants.kt
